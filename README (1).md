🎮 PokeDex - Pueblo Paleta Inc.

Esta es una aplicación web estática desarrollada en Angular que permite consultar diferentes especies de Pokémon, mostrando información como habilidades, características y datos generales obtenidos desde una API externa.

☁️ Creación de cuenta en Azure for Students

Para poder desplegar la aplicación, primero fue necesario crear una cuenta en Azure for Students. El proceso fue el siguiente:

1️- Acceso a la plataforma

Ingresé al sitio oficial de Azure for Students:
https://azure.microsoft.com/es-es/free/students/

Luego seleccioné la opción “Activar ahora”.

2️- Inicio de sesión

Utilicé mi correo institucional (por ejemplo: estudiante@universidad.edu.co) junto con la contraseña correspondiente.
Después acepté los permisos que solicita Microsoft para el uso del servicio.

3️- Verificación como estudiante

Azure verifica automáticamente que el correo pertenezca a una institución educativa válida.
Una ventaja importante es que:

No se necesita tarjeta de crédito
Se otorgan $100 USD en créditos gratuitos
4️- Activación de la suscripción

Completé mis datos personales, acepté los términos y finalicé el registro.
Luego recibí un correo confirmando la activación de la cuenta.

5️- Acceso al portal de Azure

Finalmente, ingresé al portal principal:
https://portal.azure.com

Verifiqué que la suscripción Azure for Students estuviera activa correctamente.
